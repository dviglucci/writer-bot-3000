# Writer-Bot 3000

Explore AI with Writer-Bot 3000! This friendly bot can write whatever you ask - just give it a prompt and see what it says.

Test it out for yourself at the link below:

https://writerbot3000.netlify.app/

To bring Writer-Bot 3000 to life, I used [Create React App](https://github.com/facebook/create-react-app), GPT-3 by way of the [OpenAI API](https://openai.com/api/), [Netlify](https://www.netlify.com/), and my own special sauce.
